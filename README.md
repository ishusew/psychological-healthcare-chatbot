# Psychological Healthcare Support Chatbot

CIS142-6 Natural Language Processing and Generative AI  
University of Bedfordshire — Part B, Applied Project

## Overview

A RAG-based chatbot answering psychological healthcare questions,
grounded in real counselling conversations and clinical documents.
Includes four safeguards for safe deployment with vulnerable users.

## Architecture

- **Embedding model:** all-MiniLM-L6-v2 (384-dimensional vectors)
- **Vector database:** FAISS IndexFlatL2
- **Language model:** google/flan-t5-large
- **Interface:** Gradio
- **Knowledge base:** 75 documents, 225 chunks

## Setup

1. Open `nlp_assignment.ipynb` in Google Colab
2. Enable GPU: Runtime → Change runtime type → T4 GPU → Save
3. Run all cells from top to bottom (Runtime → Run all)
4. Click the Gradio share link generated by the final cell

All libraries install automatically in Cell 0.
Clinical documents download automatically from GitHub Gist.

## Data Sources

- `fadodr/mental_health_therapy` (HuggingFace, MIT licence) — 50 conversations
- Clinical Q&A documents — 25, based on NHS and Mind.org.uk public information

## Safeguards

1. **Crisis detection** — keyword filter redirects to crisis services
2. **Topic filter** — rejects non-health queries
3. **Grounding prompt** — restricts model to retrieved context only
4. **Confidence threshold** — declines to answer when retrieval similarity is low

## Evaluation

90% accuracy across ten test queries. Mean latency 1.2 seconds.
Zero financial cost — runs entirely on Google Colab free tier.
